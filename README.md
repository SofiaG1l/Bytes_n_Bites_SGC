# Retrieving and analyzing news from the Guardian

## Introduction

In this edition of Bytes & Bites, Dr. Sofia Gil-Clavel [webpage](https://sofiag1l.github.io/) 
(researcher and manager of the Societal Analytics Lab [link](https://societal-analytics.nl/)) 
will teach how to retrieve data using Application Programming Interfaces (APIs), 
as well as how to analyze it. 

**When:** April 14th, 2026, between 16:00 and 18:00 hrs.

**Where:** Check the "Vrije Universiteit Amsterdam events & workshops" [page](https://libcal.vu.nl/event/4521460).

## Itinerary

During the session you will learn the following:

* Data retrieval and cleaning [16:00 – 16:40 hrs]:
  - How to access data via an API using the packages “dplyr” and “httr”.
  - How to scrape the data from those URLs using the R package “paperboy”.
  - How to clean the data using the package “tidytext”.
* Break [16:40 and 17:00 hrs]:
* Text as data [17:00 – 17:40 hrs]:
  - How to make sense of the data using the package “tidytext”.
  - How to visualize the results using “ggplot2” and “ggraph”.


## Requirements

Participants need to:

* Bring their own laptops.
* Have R and Rstudio installed.
* Have experience using the tidyverse packages.
* Have a Guardian developer token. The token can be requested via this webpage: [https://open-platform.theguardian.com/access/](https://open-platform.theguardian.com/access/).
